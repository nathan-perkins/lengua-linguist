# LenguaLinguist

LenguaLinguist helps you improve your accent by shadowing native speakers on YouTube. Search for a video by title or paste a YouTube link, then practice in short, repeatable segments.

## What You Can Do

- Search YouTube videos by title or keyword
- Open a YouTube link directly
- Save and return to your previous video
- Create short loop segments for shadowing practice
- Rename loop segments to keep them organized
- Record your own audio for each segment
- Drag the timeline to seek and adjust segments
- Toggle button labels for easier use

## How It Works

1. Open the app (lengualinguist.com).
2. Search for a video by typing a title, keyword, or YouTube link.
3. Select a result from the list, or open the video directly from the link.
4. Use the player controls to start loop practice.
5. Create segments you want to shadow.
6. Record yourself for each segment if you want to compare your pronunciation and rhythm.
7. Return to previous search results or switch back to your last selected video at any time.
